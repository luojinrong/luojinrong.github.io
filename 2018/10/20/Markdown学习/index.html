二、 进阶语法

1. 图片引用

语法

![引用的图片](图片地址”图片标题”)

注：“引用的图片”为显示在图片下面的文字，相当于对图片的解释，图片地址即为图片的位置，图片标题为当鼠标移到图片上时显示的内容（可加可不加）。

书写效果

![网站小图](/Markdown学习/favicon.ico “一张小图”)

显示效果

2.超链接

语法

[超链接名](超链接地址”超链接标题”)

超链接名

书写效果

[我的GitHub](http://github.com/luojinrong/)

显示效果

我的GitHub

3. 列表

  • 无序列表

    • 书写效果

      - 列表内容
      * 列表内容
      + 列表内容

      • 显示效果
  • 列表内容
  • 列表内容
  • 列表内容

注：无序列表用-+*都可以，且可以混合使用，但是-+*与内容之间要用空格隔开。

  • 有序列表

    • 书写效果

      1. 列表内容
      3. 列表内容
      5. 列表内容

    • 显示效果

      1. 列表内容
      2. 列表内容
      3. 列表内容

注：有序列表用一个数字加点实现，数字可为任意数字，显示为从一开始的有序数列，同样数字加点与内容之间需用空格隔开。

4. 表格

  • 书写效果

    |表头|表头|表头|
    |—-:|:–-\:|—–|
    |内容|内容|内容|
    |内容|内容|内容|

  • 显示效果

表头 表头 表头
内容 内容 内容
内容 内容 内容
  • 注：第二行分割表头和内容，-有一个就行，多加为了对齐（强迫症，文字默认居左，可通过在第二行加冒号调整，加在-右边居右，两边都加居中，原生语法两边都需用|括起来，现在可省略。
