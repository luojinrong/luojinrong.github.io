安装WINE

一堆废话

  • 这是一个在Linux下整出一个Windows环境的小工具，他在Linux下面映射出（暂且这么叫吧）Windows中的系统盘，然后再搭配一些动态链接库（dll），就可以运行Windows下的环境了。
  • 之前其实有看到这个工具，但是嫌麻烦没有去折腾，最近折腾数据库的时候下载了一个MySQL图形界面的工具（navicat），因为实验室电脑用的是Ubuntu，所以在官网下了Linux版的，结果下载过来，发现是wine+windows版navicat，不得不说官方十分偷懒。以至于我在使用过程中出现了乱码问题（因为Linux跟Windows编码方式不一样）。
  • 这时候想起来反正都已经下载了wine，干脆就去折腾折腾怎么用qq吧，这样就会方便今后的使用。

正式安装

  • 首先是安装deepin wine for ubuntu。（由于github下载速度过慢，整了一天没整下来，回寝室无聊看了一下readme文档发现里面写了高速下载方法（readme的重要性））
  • 这是国内的服务器，下载速度3-4MB/s，真是比我原来10-20kb快不知道多少，而且还有可能被中断。
